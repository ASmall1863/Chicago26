# Chicago ’26

Mobile-first trip quest app.

## Publish with GitHub Pages
1. Create a new public GitHub repository.
2. Upload all files in this folder, preserving `.github/workflows/pages.yml`.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **GitHub Actions**.
5. The site will appear at `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`.

On iPhone, open that URL in Safari and choose **Share → Add to Home Screen**.
