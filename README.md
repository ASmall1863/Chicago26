# Chicago ’26 — Version 2

Version 2 removes the Renee Hanover quest and adds an in-app quest editor.

## Update the live GitHub Pages app

1. Unzip this package.
2. In the GitHub repository, choose **Add file → Upload files**.
3. Drag all files from this folder into the upload area.
4. Confirm replacement of files with the same names.
5. Click **Commit changes**.
6. Wait about a minute, then refresh the app.

The app migrates existing Version 1 progress automatically while removing the deleted quest.

## New controls

- Tap **＋** to add a quest.
- Open any quest to edit or delete it.
- Export/import backups from Profile.
- Restore the default Version 2 itinerary without resetting journal progress.
